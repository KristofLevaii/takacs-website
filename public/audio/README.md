## File Requirements:
- Format: MP3 (recommended for best browser compatibility)
- File size: Keep files reasonably sized for web playback (under 10MB each recommended)
- Sample rate: 44.1kHz or 48kHz
- Bitrate: 128kbps or higher for good quality

## Notes:
- The audio files will be served from `/audio/` path in the browser
- If a file is missing, the player will show an error in the console but won't break the site
- The player will automatically loop songs when they end
- Volume controls are fully functional
